Ideabox is experimental project by Ratnadeep Bhttacharyya to build a place where creators can discuss and improve themselves and with help of community improve there projects may it be a software development, IoT, video editing, logo design, image editing are few others. At present app is very new and can be called in alpha phase with basic input and authentication and missing many features. For developers who want to know how it works please visit github site Ideabox Github and contribute to improve it. Clone it, improve it, test it and push it. Project is using firebase as backend with Angular as frontend with angular specific material styling.
